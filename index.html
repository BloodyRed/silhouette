Paramètres

Cette application fonctionne entièrement dans ton navigateur, sans serveur. Une synchronisation directe avec ces services exige une authentification OAuth côté serveur — voici les alternatives.

☁

Sauvegarde cloud Gratuit · jsonbin.io

Synchronise tes données entre plusieurs appareils grâce à jsonbin.io (gratuit). Une seule fois : crée un compte sur jsonbin.io, ouvre la page API Keys et copie ta clé X-Master-Key ci-dessous, puis clique sur « Créer mon espace ». Sur un autre appareil : colle la même clé et l'identifiant d'espace, puis « Récupérer ».

Synchro auto
Non configuré — tes données restent enregistrées sur cet appareil.
W

Withings Import CSV

Dans l'app Withings : Profil → Paramètres → Télécharger mes données. Tu recevras un fichier weight.csv — importe-le ici, toutes tes pesées seront ajoutées automatiquement (les doublons sont ignorés).

G

Google Fit Synchro directe indisponible

Google Fit exige une application enregistrée auprès de Google avec un serveur d'authentification. Tu peux exporter tes données via Google Takeout (takeout.google.com → Fit) et importer le fichier de poids en CSV ci-dessus si ton export en contient un.

S

Samsung Health Synchro directe indisponible

Samsung Health ne propose pas d'accès web ouvert. Astuce : Samsung Health peut se synchroniser avec Withings (Health Mate) via les réglages de partage — tes pesées transiteront alors par l'export Withings ci-dessus.
